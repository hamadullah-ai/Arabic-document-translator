# Arabic Scholarship Translator

AI-assisted translation tool designed to help students prepare scholarship and official academic documents in formal Arabic for Saudi university applications.

---

## Overview

This project was developed as part of independent exploration into AI-assisted language tools and practical web application development. The translator focuses on formal academic and government-style Arabic suitable for scholarship-related documents.

---

## Features

- English to Arabic document translation
- Multiple academic document modes
- Formal Modern Standard Arabic output
- Responsive user interface
- Copy-ready translated output

---

## Technologies

- React.js
- JavaScript
- Claude API
- HTML/CSS
- Netlify
